# 📊 Sales Performance Dashboard (Power BI)

## 🔍 Project Overview
This project analyzes sales performance using Power BI to uncover insights on revenue, profit, and customer trends.

## 📌 Key Metrics
- Total Sales
- Total Profit
- Total Orders

## 📊 Dashboard Insights
- Sales trend over time
- Top-selling products
- Sales distribution by category
- Regional performance analysis

## 🛠 Tools Used
- Power BI
- Excel
